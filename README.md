This repository, 15-darn-sql-injection, documents a comprehensive security analysis and technical walkthrough of SQL Injection vulnerabilities within the Damn Vulnerable Web Application (DVWA). Throughout this assignment, I transitioned through Low, Medium, and High security configurations to demonstrate how different defensive layers—ranging from raw unsanitized inputs to client-side dropdown restrictions and out-of-band input windows—can be systematically bypassed. By employing a variety of exploitation techniques including UNION-based attacks, HTML manipulation via browser developer tools, and Boolean-based blind injection, this project highlights the critical inadequacy of "security through obscurity" and emphasizes the necessity of server-side remediation through prepared statements and parameterized queries.
